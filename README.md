# Livelong AI (Livelong AL) 🩺📈🛡️

An advanced, AI-powered corporate insurance and underwriting analytics dashboard designed to optimize customer profiles, lead tracking, claim evaluation, and agent operations.

Built with a modern React + TypeScript frontend, Livelong AI offers insurance brokerages and carriers real-time insights, compliance tracking, and interactive AI agent copilot support.

---

## ✨ Features

- **📊 Underwriting & Revenue Dashboards**: Analytical boards displaying commission logs, active insurance volumes, and revenue run-rates using Recharts.
- **🤖 AI Insurance Copilot**: Interactive assistant built with conversational UI to help agents research underwriting rules, drafts, and customer summaries.
- **🛡️ Policies & Claims management**: Modules to audit compliance scores, verify claim details, track coverage status, and log renewals.
- **📈 CRM & Leads Pipeline**: Visual pipelines for managing new insurance prospects, agent tasks, and automated marketing segments.
- **💼 Brokerage & Branch Coordinator**: Coordination systems mapping customer databases, broker details, and document updates across local branches.

---

## 🛠️ Tech Stack

- **Framework**: React 18 & Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS & Custom Components
- **Charts & Visualizations**: Recharts
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sarang-24/livelong-al.git
   cd livelong-al
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
